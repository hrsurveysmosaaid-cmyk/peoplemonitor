Place Arabic-capable TrueType fonts here for PDF generation.

Recommended:
- NotoNaskhArabic-Regular.ttf
- NotoNaskhArabic-Bold.ttf

Download from the official Google Fonts repository:
https://github.com/googlefonts/noto-fonts/tree/main/hinted/ttf/NotoNaskhArabic

After adding the files, re-run the server. The PDF exporter will automatically use these fonts and fix garbled Arabic text in section headings and content.
